## HireXpert — Job Finder Mobile App

A modern React Native job search application that helps users discover remote jobs, save favorites, and manage their professional profile — all in one place.

## Features

✔ User Authentication (Login / Signup)
✔ Browse Remote Jobs (Live API)
✔ Save Jobs for Later
✔ Apply Button UI
✔ Profile Management
✔ Resume Upload
✔ Profile Image Picker
✔ Clean & Modern UI

## Tech Stack

✔ React Native
✔ Expo
✔ React Navigation
✔ Context API (State Management)
✔ Remotive Jobs API
